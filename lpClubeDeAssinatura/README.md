# Encantos Literarios

Neste desafio, proposto pela Rocketseat, desenvolvi uma Landing Page para um clube de assinatura literario chamado `Encantos Literarios`.

O objetivo da pagina e apresentar o clube de forma visualmente atrativa, destacar os beneficios da assinatura, mostrar um exemplo do kit recebido pelo assinante e exibir os planos disponiveis.

## Sobre o projeto

A landing page foi construida com:

- `HTML`
- `CSS`
- `JavaScript`

O projeto foi desenvolvido com foco em:

- apresentacao visual do clube
- interacoes com hover e animacoes
- destaque dos planos de assinatura
- responsividade para desktop e mobile
- melhorias de acessibilidade

## Estrutura principal

Os arquivos principais do projeto sao:

- `index.html`: estrutura da pagina
- `style.css`: estilos, layout, responsividade e animacoes
- `script.js`: interacoes dos cards de assinatura

## Como executar

Como este e um projeto estatico, nao e necessario instalar dependencias.

Voce pode visualizar de duas formas:

### 1. Abrindo diretamente no navegador

Basta abrir o arquivo:

- `index.html`

### 2. Usando a extensao Live Server no VS Code

Se estiver usando o VS Code:

1. instale a extensao `Live Server`
2. clique com o botao direito em `index.html`
3. escolha a opcao `Open with Live Server`

Assim a pagina sera aberta no navegador com atualizacao automatica a cada alteracao.

## O que a pagina apresenta

- secao inicial com chamada principal
- secao descritiva com destaque para os beneficios
- secao do kit mensal com elementos visuais interativos
- secao de assinatura com tres planos
- footer com links e redes sociais

## Observacoes

- o projeto utiliza imagens e icones locais dentro da pasta `imagens`
- algumas interacoes visuais dependem de `hover`
- os cards da assinatura possuem comportamento interativo com JavaScript

## Visualizacao

Para visualizar corretamente:

- abra o projeto em um navegador moderno
- mantenha a estrutura de pastas original
- garanta que a pasta `imagens` esteja no mesmo nivel dos arquivos principais
